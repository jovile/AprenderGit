Aprender Git y GitHub con Visual Studio Code

Este sitio es un ejemplo práctico y didáctico para aprender a usar git y GitHub desde Visual Studio Code.

¿Qué verás aquí?

  • Archivos del proyecto (HTML, CSS, JS).
  • Comandos básicos de git para inicializar, commitear y subir al remoto.
  • Guía de uso de la interfaz Git de VS Code y la extensión de GitHub.

Comandos básicos

git init -b main
git add .
git commit -m "Initial scaffold: Aprender Git con VSC"
git remote add origin https://github.com/USERNAME/REPO.git
git push -u origin main

Actividad rápida

Pulsa el botón para ver un mensaje simulado de "commit" en la consola.
